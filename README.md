## Installation
Attribute Authority is available on [PyPI](https://pypi.org/project/attribute_authority/). Install using `pip`:
```
pip install attribute_authority
```

You can also install from the git repository:
```
git clone https://github.com/hmffa/attribute_authority
pip install -e ./attribute_authority
```
